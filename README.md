# Mason Picks

Find the easiest courses and highest GPA professors at George Mason University. A searchable database to help students boost their GPA.

## Features
- **Search**: Filter courses by code, title, or professor.
- **Filter**: Filter by Department/Subject and Minimum GPA.
- **Sort**: Sort by GPA, Popularity (Class Size), or Instructor.
- **Responsive**: Works on desktop and mobile.

## Data Source
Data is sourced from George Mason University via FOIA requests.
*Disclaimer: This project is not officially affiliated with George Mason University.*

## Development

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Process Data**:
    The app uses a JSON file generated from the CSV.
    ```bash
    npm run process-data
    ```

3.  **Start Server**:
    ```bash
    npm run dev
    ```
